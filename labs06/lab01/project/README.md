# Лабораторная работа 6.1 - Haskell

## Структура проекта

- `basics.hs` - базовые функции
- `recursion.hs` - рекурсивные функции
- `patterns.hs` - pattern matching
- `higher_order.hs` - функции высшего порядка
- `types.hs` - алгебраические типы данных
- `practice.hs` - практические задания
- `main.hs` - главный файл для запуска

## Компиляция и запуск

```bash
ghc -o haskell_demo main.hs
./haskell_demo
```

Или через интерпретатор:
```bash
ghci
:load main.hs
main
```
