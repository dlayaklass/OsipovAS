Введение

Данный документ создан с использованием технологии R Markdown для демонстрации возможностей грамотного программирования. В документе представлены различные расчеты и визуализации данных.

Теоретические сведения

Грамотное программирование (literate programming) - это методология программирования, предложенная Дональдом Кнутом, которая позволяет создавать программы, где код и документация объединены в единый документ.

Данные и расчеты

Создание тестовых данных

# Создание вектора случайных чисел
set.seed(123)
random_data <- rnorm(100, mean = 50, sd = 10)

# Вычисление статистических характеристик
mean_value <- mean(random_data)
median_value <- median(random_data)
sd_value <- sd(random_data)
min_value <- min(random_data)
max_value <- max(random_data)

Статистические характеристики

Среднее значение данных составляет 50.90, медиана равна 50.53, а стандартное отклонение - 9.87.

Минимальное значение: 26.91, максимальное: 71.69.

Визуализация данных

# Создание гистограммы
hist(random_data, 
     main = "Распределение случайных данных",
     xlab = "Значения",
     ylab = "Частота",
     col = "lightblue",
     border = "black",
     breaks = 20)

# Добавление линии среднего значения
abline(v = mean_value, col = "red", lwd = 2, lty = 2)
text(mean_value, 5, "Среднее", col = "red", pos = 4)

Работа с матрицами

# Создание матрицы
matrix_data <- matrix(1:20, nrow = 4, ncol = 5)
colnames(matrix_data) <- paste("Столбец", 1:5)
rownames(matrix_data) <- paste("Строка", 1:4)

# Вычисление суммы по строкам
row_sums <- apply(matrix_data, 1, sum)

# Вычисление суммы по столбцам
col_sums <- apply(matrix_data, 2, sum)

Сумма элементов матрицы: 210.

Заключение

Данный документ демонстрирует возможности R Markdown для создания динамических отчетов, объединяющих код, результаты вычислений и текстовое описание в единый документ.

Основные возможности, продемонстрированные в документе:

  • Inline код (встроенный в текст)
  • Code chunks (блоки кода)
  • Различное форматирование текста
  • Визуализация данных
  • Таблицы результатов
  • Использование CSS для стилизации
