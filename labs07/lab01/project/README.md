# Лабораторная работа: Асинхронное программирование в Go

## Описание
Демонстрация работы с горутинами, каналами, worker pool и HTTP сервером в Go.

## Структура проекта
- `cmd/main.go` - основная программа
- `internal/async/` - модули для работы с горутинами и каналами
- `internal/server/` - HTTP сервер

## Запуск
```bash
go run cmd/main.go
```

## Компоненты
1. Базовые горутины с WaitGroup
2. Каналы (буферизованные и небуферизованные)
3. Worker Pool паттерн
4. HTTP сервер с graceful shutdown
