# Лабораторная работа: Тестирование асинхронного кода в Go

## Описание
Тестирование компонентов асинхронного программирования: горутины, каналы, worker pool и HTTP сервер.

## Структура проекта
- `internal/async/` - модули и тесты для работы с горутинами и каналами
- `internal/server/` - HTTP сервер и тесты

## Запуск тестов
```bash
# Все тесты
go test ./...

# С детектором гонок
go test -race ./...

# Покрытие кода
go test -cover ./...
```

## Компоненты
1. Тесты для Counter и ProcessItems
2. Тесты для каналов
3. Тесты для Worker Pool
4. Тесты для HTTP сервера
