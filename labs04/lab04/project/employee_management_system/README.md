# Система учета сотрудников компании

Комплексная система учета сотрудников с применением принципов ООП, композиции и агрегации.

## Структура проекта

```
employee_management_system/
├── src/                 # Исходный код системы
│   ├── core/           # Основные классы
│   ├── employees/      # Классы сотрудников
│   ├── factories/      # Фабрики
│   └── utils/          # Вспомогательные модули
├── examples/           # Примеры использования
├── data/               # Данные (JSON, CSV)
└── docs/               # Документация
```

## Запуск

```bash
python main.py
```

или

```bash
python examples/demo_part4.py
```

## Основные возможности

- Управление сотрудниками, отделами и проектами
- Экспорт данных в CSV и JSON
- Анализ данных и статистика
- Планирование и назначение сотрудников на проекты
- Сериализация и десериализация всей системы
